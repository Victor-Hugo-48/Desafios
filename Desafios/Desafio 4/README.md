## Descrição
Este projeto apresenta a resolução de exercícios práticos sobre matrizes em Python, conforme proposto em aula.

Foram aplicados conceitos básicos de programação, como:
- Estruturas de repetição (for)
- Listas (matrizes)
- Operações matemáticas
- Uso de bibliotecas (`random` e `numpy`)

## Exercícios Desenvolvidos

### Exercício 1
Criação de uma matriz 3x3 com números aleatórios.

### Exercício 2
Soma de duas matrizes representando vendas.

### Exercício 3
Cálculo da média de alunos utilizando NumPy.

### Exercício 4
Cálculo do determinante de uma matriz e verificação de solução.

### Exercício 5
Transposição e multiplicação de matrizes.

## Tecnologias Utilizadas
- Python 3
- Biblioteca `random`
- Biblioteca `numpy`

## Como executar
1. Instalar o Python
2. Instalar o NumPy: pip install numpy
3. Executar o arquivo: python matrizes_exercicios.py

## Conclusão
O trabalho permitiu praticar a criação e manipulação de matrizes em Python, além de compreender a diferença entre o uso manual e o uso de bibliotecas como o NumPy.

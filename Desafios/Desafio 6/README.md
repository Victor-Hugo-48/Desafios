## Sobre o desafio
Este projeto foi desenvolvido com o objetivo de praticar a resolução de sistemas de equações lineares utilizando Python e a biblioteca NumPy.

Os exemplos e exercícios simulam situações simples do dia a dia, ajudando a entender como a matemática pode ser aplicada na programação.

## Tecnologias utilizadas
- Python 3  
- Biblioteca NumPy (`np.linalg.solve`)  

## Organização
O projeto está dividido em:

- **Desafio** → resolução principal do sistema  
- **Exemplos** → exercícios resolvidos para aprendizado  
- **Exercícios** → aplicações práticas  

## Como executar
1. Instale o NumPy:
pip install numpy

2. Execute um arquivo:
python Exercicio1.py

## Como funciona
O programa utiliza matrizes para resolver os sistemas de equações.  
Após encontrar os valores, é feita uma verificação substituindo nas equações originais para confirmar o resultado.

## Observação
Caso a matriz não possa ser resolvida (determinante igual a zero), o programa informa que o sistema não possui solução única.

## Conclusão
Este projeto ajudou a reforçar conceitos de lógica, matemática e programação, mostrando na prática como resolver problemas usando código.

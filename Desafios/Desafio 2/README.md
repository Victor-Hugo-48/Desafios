# Desafio 2 - Teoria de Listas em Python

Este projeto consiste num script educativo desenvolvido para demonstrar os conceitos fundamentais de manipulação de listas na linguagem Python, conforme solicitado no roteiro de atividades da disciplina.

## Conceitos Abordados

O script `desafio2_listas.py` contém codigos e comentarios sobre:

1. **Criando listas**: Criação de coleções ordenadas e mutáveis.
2. **Acesso e Fatiamento**: Uso de índices positivos, negativos e extração de sublistas.
3. **Iteração**: Percorrimento de dados utilizando estruturas de repetição (`for` e `while`).
4. **Operações com listas**: Métodos para adicionar (`append`, `insert`), remover (`remove`, `pop`) e alterar elementos.
5. **Estrutura de Pilha (LIFO)**: Implementação do conceito *Last In, First Out* utilizando métodos nativos.
6. **Operações de Ordenação**: Uso de `sort()` e `reverse()` para organização de dados.
7. **Listas Aninhadas**: Representação de matrizes e acesso a múltiplos níveis.
8. **Compreensão de Listas**: Sintaxe avançada e concisa para criação de novas listas.

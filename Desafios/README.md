# Desafio 1 - Sistema de Folha de Pagamento

## Descrição
Programa em Python para gerenciar a folha de pagamento de uma pequena empresa.  
Permite cadastrar funcionários dos tipos Estagiário, CLT e Freelancer, calcular salários líquidos, gerar relatório e salvar em arquivo de texto.

## Funcionalidades
- Cadastro de funcionários (Estagiário, CLT e Freelancer)
- Cálculo automático de descontos (INSS e IRRF)
- Geração de relatório no console
- Salvamento do relatório em arquivo .txt
- Validações de entrada (nome não vazio, tipos válidos, valores positivos)
- Tratamento de erros

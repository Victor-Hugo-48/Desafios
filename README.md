# Desafio - Sistema de Folha de Pagamento

## Descrição
Este projeto foi desenvolvido como desafio da Aula 2 da disciplina de Programação.  
O sistema gerencia a folha de pagamento de uma pequena empresa, permitindo o cadastro de funcionários dos tipos Estagiário, CLT e Freelancer, cálculo dos salários líquidos e geração de relatório.

## Funcionalidades
- Cadastro de funcionários (Estagiário, CLT e Freelancer)
- Cálculo automático do salário líquido seguindo as regras do enunciado
- Geração de relatório detalhado no console
- Salvamento do relatório em arquivo `.txt`
- Validações de entrada e tratamento de erros
